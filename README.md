# NN-from-scratch
Creating Neural Network from Scratch with Numpy (multi-class classification)

## Essestials
* Neural Network model
    * Linear layer
    * Activation function
        * ReLU
        * Sigmoid

* Loss function
    * Cross Entoropy
    
* Optimizer
    * SGD (with Momentum)

## Run
* requirements
    * python >= 3.6
    * numpy == 1.18.1 (ambiguous)
```bash
$ python main.py
```
